# 3D 动画与场景优化项目 - 启动与操作指南

本说明文档将引导您完成项目的本地环境搭建、启动流程以及核心功能的运行说明。

## 1. 系统要求与环境准备

*   **Node.js**: 推荐版本 v18.x 或更高（本项目使用 Vite 驱动）。
*   **浏览器**: 推荐使用最新版的 Chrome、Edge 或 Firefox（需支持 WebGL 2.0）。
*   **显卡**: 建议具备独立显卡或高性能核显，以获得流畅的 3A 级环境渲染体验。

## 2. 快速启动流程

请按照以下顺序在终端执行命令：

### 第一步：安装依赖项
在项目根目录（包含 `package.json` 的目录）执行：
```bash
npm install
```

### 第二步：启动开发服务器
由于项目结构原因，建议使用以下命令在 `src` 目录下启动并指向父级根目录：
```bash
# 在 d:\workspace\test\src 目录下执行
npx vite .. --port 5173 --host
```

### 第三步：访问游戏
在浏览器中打开：`http://localhost:5173/`

## 3. 动画与运动系统说明 (3A 标准优化)

为了提升角色运动的自然度和打击感，代码已通过以下技术手段进行了深度优化：

1.  **拟真步态 (Locomotion)**: 
    *   引入了 **Hip Sway (胯部摆动)**：角色走路时会有自然的重心偏移。
    *   **Body Bobbing (上下起伏)**：模拟行走时的重力受压与弹起。
    *   **Shoulder Counter-rotation (肩膀补偿旋转)**：上身与下身呈相反相位摆动，增加灵动感。
    *   **Foot IK Offset (足部偏移增强)**：修复了脚底滑步，增加了落地时的缓冲。

2.  **技能释放优化**:
    *   **Anticipation (预备姿态)**：技能释放瞬间增加 0.1s 的聚气微调，增强爆发感。
    *   **Screen Shake (屏幕震动)**：R 技能落地伴随强力震屏，增强打击反馈。
    *   **Particle Trails (粒子拖尾)**：所有技能均添加了动态能量粒子和光影效果。

## 4. 操作说明

| 键位 | 功能描述 |
| :--- | :--- |
| **W / A / S / D** | 角色移动 |
| **鼠标移动** | 调整相机视角 |
| **空格 (Space)** | 跳跃 |
| **Q 键** | 技能 1：水流冲刺 (Dash) |
| **W 键** | 技能 2：旋风斩 (Spin Attack) |
| **E 键** | 技能 3：护盾 (Shield) |
| **R 键** | 技能 4：终极一击 (Slam / Ultimate) |
| **鼠标左键** | 基础攻击 (含多段粒子特效) |

## 5. 常见问题排查

1.  **404 错误**: 确保启动命令中的根路径指向正确（`index.html` 所在的目录）。
2.  **画面掉帧**: 请检查浏览器是否开启了“硬件加速”。
